Website ini dibuat menggunakan HTML saja dengan struktur dasar seperti doctype, html, head, dan body biar halaman punya kerangka yang jelas. 
Di bagian head, dipakai meta charset supaya teks tampil normal dan title untuk judul tab browser. 
Seluruh isi web diletakkan di dalam body dan disusun pakai heading, paragraf, teks tebal, serta garis pemisah agar tampilan halaman lebih rapi dan gampang dibaca.

Biar pengguna nggak bingung saat berpindah bagian, website ini dilengkapi navigasi sederhana menggunakan link yang mengarah ke section tertentu dalam satu halaman. 
Setiap bagian punya id sendiri, jadi saat link diklik, halaman langsung scroll ke bagian yang dituju. 
Beberapa konten seperti keahlian dan hobi juga dibuat bisa dibuka-tutup pakai details dan summary, biar kelihatan lebih interaktif meskipun tanpa JavaScript.

Untuk menampilkan data dan media,website ini menggunakan berbagai elemen HTML seperti tabel <table>, <tr>, <th>, <td> untuk menyajikan riwayat pendidikan, list <ul>, <ol>, <li>
untuk keahlian, hobi, fun facts, dan game favorit, serta gambar profil dengan tag <img>.
Selain itu, musik favorit ditampilkan melalui video YouTube yang di-embed menggunakan tag <iframe>, sehingga konten multimedia dapat ditampilkan langsung di dalam halaman web.

Di bagian akhir, website ini dilengkapi form kontak sederhana sebagai latihan penggunaan form di HTML, yang berisi input nama, email, pesan, dan tombol kirim. Ada juga link ke GitHub dan Instagram sebagai informasi tambahan.
Secara keseluruhan, website ini dibuat tanpa framework, CSS, atau JavaScript, biar fokus utamanya tetap ke pemahaman dasar HTML dan penyusunan halaman web statis.
