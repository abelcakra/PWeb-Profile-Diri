Website ini dibuat menggunakan HTML saja dengan struktur dasar seperti doctype, html, head, dan body biar halaman punya kerangka yang jelas. Di bagian head, digunakan meta charset supaya teks bisa tampil normal serta title untuk judul tab browser. Seluruh isi website diletakkan di dalam body dan disusun menggunakan heading, paragraf, teks tebal, serta garis pemisah agar tampilannya lebih rapi dan gampang dibaca.

Biar pengguna nggak bingung saat berpindah bagian, website ini dilengkapi dengan navigasi sederhana menggunakan link yang mengarah ke section tertentu dalam satu halaman. Setiap bagian punya id sendiri, jadi saat link diklik, halaman akan langsung scroll ke bagian yang dituju. Beberapa konten seperti keahlian dan hobi juga dibuat bisa dibuka dan ditutup menggunakan elemen details dan summary, biar kelihatan lebih interaktif meskipun tanpa JavaScript.

Untuk menampilkan data dan media, website ini menggunakan berbagai elemen HTML seperti tabel untuk menyajikan riwayat pendidikan, list untuk menampilkan keahlian, hobi, fun facts, dan game favorit, serta gambar untuk foto profil. Selain itu, musik favorit ditampilkan melalui video YouTube yang di-embed ke dalam halaman, sehingga konten multimedia bisa langsung ditampilkan tanpa harus membuka halaman lain.

Di bagian akhir, website ini dilengkapi dengan form kontak sederhana sebagai latihan penggunaan form di HTML, yang berisi input nama, email, pesan, dan tombol kirim. Terdapat juga link menuju GitHub dan Instagram sebagai informasi tambahan. website ini dibuat tanpa framework, CSS, maupun JavaScript, biar fokus utamanya tetap pada pemahaman dasar HTML dan penyusunan halaman web statis.
